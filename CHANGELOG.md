# CHANGELOG

## V1.3.0 (2026-07-16)

### 语录页面
- 画廊式卡片网格布局，12种主题色标签颜色系统
- 标签筛选 + 时间筛选（年/月多选）
- 点击卡片弹窗详情查看（两端对齐排版 + 首行缩进）
- 每次进入页面随机展示语录

### 日志页面
- 双栏布局（左30%筛选+卡片，右70%预览+编辑）
- 白色网格卡片（心情emoji + 内容摘要）
- 日期筛选盒（年份下拉 + 月份圆形按钮多选）
- 心情标签筛选 + 自定义心情管理
- 移动端弹窗适配

### 物品台账页面
- 分类改标签系统（12色匹配 + 标签管理 + 换色）
- 图片比例统一4:3，标签显示在图片左上角
- 增加物品弹窗优化（退役开关、标签chip选择）
- 物品排序（未退役优先 + 最新在前）
- 搜索框隐藏为图标

### 全局优化
- 移动端响应式适配（边距/卡片/双栏堆叠）
- 弹窗时锁定背景页面滚动
- 数据导入支持覆盖/合并两种模式
- imageManager 图片URL内存缓存

### Bug修复
- 编辑作品保留点赞评论数据
- 新增日志不再重复保存
- 首页评论折叠优化（>10条折叠）
- 首页封面更换改为图标+悬浮菜单（支持还原封面）

## V1.2.0 (2026-07-14)

### 画廊交互升级
- 点赞功能：记录姓名+时间，每日限一次（零点为界）
- 评论功能：记录姓名+时间+内容，评论人可手动修改
- 作品画廊分类标签自定义（新增/编辑/删除，重命名自动同步）
- 作品创建时间精确到时分，卡片右下角显示
- 画廊卡片按创建时间由新到旧排列

### 体验优化
- 首页评论默认展开显示
- 图片URL内存缓存 + 并行加载，解决页面切换卡顿
- 画廊卡片描述改为单行省略
- 点赞/评论图标移至卡片右下角
- 编辑作品不再丢失已有点赞评论数据
